For å kjøre prosjektet går du inn i DemoApplication.java.

Dette er en enkel html side med en Java-backend konfigurert med SpringBoot rammeverket.

Java backenden har to Get-apier, som henter brukere og spørsmål.

Frontend består av Html og Javascript, med funksjonalitet for å trykke seg igjennom spørsmålene og få tilbakemelding om riktig eller feil svar. 

Brukeren kan også trykke på submit quiz, og få opp sin poengsum og andre brukere.
